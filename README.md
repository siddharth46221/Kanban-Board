## Installation & Usage

1. **Clone the repository**
   ```sh
   git clone https://github.com/your-username/offline-kanban-board.git
   ```
2. **Open the project folder**
   ```sh
   cd offline-kanban-board
   ```
3. **Open `index.html` in a browser** (No setup required, as it works offline).

## How It Works

1. Add tasks to the "To-Do" column.
2. Drag tasks between columns to track progress.
3. Click a task to edit or delete it.
4. Changes are automatically saved in LocalStorage.
5. Toggle dark mode for a better visual experience.
